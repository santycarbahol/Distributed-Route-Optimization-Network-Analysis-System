# Distributed-Route-Optimization-Network-Analysis-System

This project demonstrates:
- Graph-based route optimization using Dijkstra and A*
- Low-level UDP socket communication
- Virtual memory replacement simulation (FIFO, LRU)
- Modular, testable, production-quality Python code

### How to Run
1. `python server/server.py`
2. `python client/client.py`
